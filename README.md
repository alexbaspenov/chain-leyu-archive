# chain-leyu-archive

## 项目简介

chain-leyu-archive 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 文件的集合存储与展示空间。

## 目录结构

```
chain-leyu-archive/
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面引用的资源文件（图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/` 目录下存放各独立 HTML 页面，按归档时间或主题命名。
- `assets/` 目录存放页面所需的公共资源，建议按类型子目录组织。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，每个页面文件可单独访问。页面内容可能包含文本、样式、交互脚本等，但均不指向外部具体域名或服务。归档页面主要用于资料保存与历史查阅，不保证长期可用或内容更新。
